## 🧑‍🚀 Getting started

Install dependencies

```sh
npm install
```

## 🚀 Running app

```sh
npm run build
```

Or with number of days as args:

```sh
npm run build -- 10
```

## 🧪 Running tests

To run all tests

```sh
npm run test
```

👁️ To run all tests in watch mode

```sh
npm run test:watch
```
